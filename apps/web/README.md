# Web App

Next.js 14+ App Router application for the Society of Catholic Scientists website.

## Development

Run the development server:

```bash
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) to view the app.

## Testing

Run unit tests:

```bash
pnpm test
```

Run e2e tests:

```bash
pnpm e2e
```

## Building

Build for production:

```bash
pnpm build
pnpm start
```
